üê≥ HOW TO RUN WITH DOCKER

Make sure you have Docker and Docker Compose installed.
MongoDB Atlas will still be used (no local DB container needed).

Step 1: Clone the repository
git clone repo_url
cd folder_name

Step 2: Verify Backend .env

Inside the /backend folder, ensure you have a .env file with:

MONGODB_URI=your-mongodb-atlas-connection-string
JWT_SECRET=your-secret-key


‚úÖ Docker Compose automatically loads this environment file.

Step 3: Build and Start Containers

From the project root, run:
